Experimental new interface for [Storj DCS](https://www.storj.io/).

## Usage

```
git clone git@github.com:storjrd/iris.git --recursive
cd iris
```

> Requires `just`, `docker`, and `docker-compose`

```
just
```

### Development

```
just dev
```
